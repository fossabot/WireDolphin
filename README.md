WireDolphin
======

使用 QT6 制作的简易 WireShark

## Feature List

- [x] 基本框架界面
- [x] 表格和树界面可改变宽度
- [x] libpcap 抓包
- [x] 优化 libpcap 性能
- [x] 1000M 网络不丢包
- [x] 二层协议解析
- [ ] 二层协议详情树
- [ ] 三层协议详情树
- [ ] 四层协议详情树
- [ ] 表格根据协议内容按不同颜色展示
- [ ] ARP、ICMP、HTTP、DNS 解析
- [ ] HTTPS client hello 识别
- [ ] 识别 TCP Flag、Seq、NextSeq、Window 等
- [ ] 识别 TCP 分段丢失、乱序、重传
- [ ] 识别 TCP、HTTP 流
- [ ] 网卡、会话、IP维度的统计能力
- [ ] 表格组件性能问题（目前瞬间70万数据界面会无响应）
- [ ] 导入 pcap 分析
- [ ] 数据包重放
